Game Rules

1. Each player takes turns to say a word from the selected category.

2. The word must start with the current letter.

3. No repetitions allowed.

4. If a player cannot think of a word, they are out!

5. The last player standing wins!

6. The timer set at the beginning cannot be changed when game starts.

7. Number of players set at the beginning cannot be changed mid game except you end the current game session.

8. You can only use the Stop button when all inputs in your panel are filled.
